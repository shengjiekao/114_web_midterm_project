
# 專案名稱
- 智慧財產權資訊與線上諮詢平台

### 組員
- 許紘瑋
- 高聖傑

### 專案介紹
- 本專案是一個提供智慧財產權（專利、商標、著作權、營業秘密）資訊服務的網站。
使用者可以透過網站閱讀法條和透過律師介紹、了解律師專長、瀏覽事務所流程說明，並選擇喜歡的律師進行線上諮詢預約。
網站整體以 Bootstrap 建立響應式設計，並搭配 JavaScript 進行互動，包括：
1. 律師介紹彈跳視窗
2. 法規下載彈跳視窗
3. 服務流程分步切換動畫
4. 表單驗證
5. LocalStorage 表單保存
6. 回到頂端按鈕

### 使用技術
- HTML
- CSS
- Bootstrap
- JavaScript
- Constraint Validation API
- localstorage

### 功能特色
  1. 彈跳視窗
        使用者點擊卡片後，會顯示相關的資訊。

  2. 法規下載模態視窗
        每個服務領域卡片都綁定對應的 PDF 法規檔案。

  3. 線上諮詢表單
        使用 Bootstrap 驗證與 Constraint Validation API
        自動記錄每個欄位輸入狀態（localStorage）

  4. 服務流程分步切換
        採步驟切換
        進度條動態更新
     
  5.back to top button   

  6. RWD
